Fidelis Endpoint Desktop Alert Notification

 

 

API Settings:

Web Server – This is the server where the website is installed.

User Name – This is the user that will log into the Fidelis Endpoint Application to receive alerts. This user must have API privileges in the application.

Password – The password for the user that is logging in.

 

 

Alert Configuration:

Color Scheme – Select a scheme of your choosing to change how the notifications look.

Alert Display Time – The amount of time the alert notification stays in the bottom right corner before disappearing.

Time Between Checks – This tells the application how long to wait between checks for new alerts.

Minute Window for Query – Amount of time in the past to check alerts from last check time. This makes sure you do not miss any alerts that may have come in.

Sound on Alert – Turns sound on and off for alert notifications.

Re-create Database – Recreates the database that the alerts are stored in.
